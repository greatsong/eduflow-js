# 자동 조명 만들기

**Part 5**: 세상을 감지하라 - 센서 활용

---

## 학습 목표

- [ ] 센서 값을 기준으로 조건문을 작성할 수 있다
- [ ] 임계값(threshold)의 개념을 이해하고 적절히 설정할 수 있다
- [ ] 어두워지면 자동으로 켜지는 조명을 완성할 수 있다

## 개요

센서 값이 특정 기준(임계값)보다 낮으면 어둡다고 판단하고 LED를 켜는 프로그램을 만듭니다. 현관등이나 가로등이 바로 이 원리로 동작해요!

임계값을 너무 낮게 설정하면 아주 어두울 때만 켜지고, 너무 높게 설정하면 밝은 대낮에도 켜집니다. 직접 실험하면서 우리 교실에 맞는 적절한 값을 찾아봅시다. 조명 밝기도 단계별로 조절할 수 있다면 더 스마트한 조명이 되겠죠?

## 예상 소요 시간

1시간
