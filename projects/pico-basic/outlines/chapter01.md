# 피코야, 반가워! - 하드웨어 살펴보기

**Part 1**: 피코와 첫 만남

---

## 학습 목표

- [ ] 라즈베리파이 피코의 역할과 구성 요소를 설명할 수 있다
- [ ] 피코 보드의 주요 핀과 포트의 위치를 파악할 수 있다
- [ ] 피코를 컴퓨터에 안전하게 연결할 수 있다

## 개요

피코는 우리가 만들 모든 프로젝트의 '두뇌' 역할을 합니다. 스마트폰이 앱을 실행하듯, 피코는 우리가 작성한 코드를 실행해서 LED를 켜고, 소리를 내고, 센서 값을 읽어옵니다. 이번 챕터에서는 키트에 포함된 피코 보드를 직접 손에 들고 살펴보면서 각 부분의 이름과 역할을 알아봅니다.

보드 위의 작은 칩들이 무슨 일을 하는지, USB 포트는 왜 필요한지, 옆에 늘어선 핀들은 어디에 쓰이는지 하나씩 확인해 봅시다. 마치 새 스마트폰을 샀을 때 박스를 열고 이것저것 살펴보는 것처럼요!
