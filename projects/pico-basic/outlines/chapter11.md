# 나만의 멜로디 - 학교종 연주하기

**Part 4**: 소리로 표현하기 - 부저 활용

---

## 학습 목표

- [ ] 음계와 주파수의 관계를 이해할 수 있다
- [ ] 악보를 코드로 변환하는 방법을 익힐 수 있다
- [ ] 함수를 만들어 코드를 깔끔하게 정리할 수 있다

## 개요

간단한 악보를 코드로 바꿔서 피코가 연주하게 만들어봅시다! '학교종'이나 '비행기' 같은 동요는 음이 간단해서 첫 번째 연주곡으로 딱입니다.

각 음표의 주파수와 길이를 리스트에 저장하고, 반복문으로 순서대로 재생하면 멜로디가 완성됩니다. 여기서 함수의 개념도 배웁니다. 자주 사용하는 코드를 함수로 만들어두면 더 쉽게 새로운 노래를 추가할 수 있어요.
