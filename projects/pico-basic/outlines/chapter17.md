# 만들기 - 회로 구성과 코딩

---

## 학습 목표

- [ ] 설계서를 바탕으로 회로를 구성할 수 있다
- [ ] 여러 기능을 하나의 프로그램으로 통합할 수 있다
- [ ] 발생하는 문제를 스스로 해결하는 디버깅 능력을 기를 수 있다

## 개요

기획서를 바탕으로 실제 회로를 구성하고 코드를 작성합니다. 여러 기능을 하나로 합치다 보면 예상치 못한 문제가 생길 수 있습니다. 당황하지 말고 하나씩 확인해가며 해결해봅시다.

막히는 부분이 있으면 이전 챕터를 다시 살펴보거나, 친구와 상의하거나, 선생님께 질문하세요. 문제를 해결하는 과정에서 가장 많이 배웁니다. 오류 메시지를 읽는 방법, 코드를 조금씩 테스트하는 방법 등 디버깅 기술도 익혀봅시다.
