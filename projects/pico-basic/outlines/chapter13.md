# 밝기 감지 - 조도 센서

**Part 5**: 세상을 감지하라 - 센서 활용

---

## 학습 목표

- [ ] 아날로그 입력과 디지털 입력의 차이를 설명할 수 있다
- [ ] ADC(아날로그-디지털 변환)의 개념을 이해할 수 있다
- [ ] 조도 센서 값을 읽어 밝기를 측정할 수 있다

## 개요

버튼은 눌렀다/안 눌렀다 두 가지 상태만 있지만, 밝기는 0부터 100까지 다양한 값을 가집니다. 이런 연속적인 값을 읽으려면 아날로그 입력이 필요합니다.

조도 센서(CdS 또는 포토 레지스터)를 연결하고 값을 읽어봅시다. 손으로 가리면 값이 변하고, 스마트폰 손전등을 비추면 또 달라집니다. 이 값을 활용하면 자동으로 켜지는 조명을 만들 수 있어요!
