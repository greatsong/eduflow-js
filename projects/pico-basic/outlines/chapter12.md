# 피아노 만들기 - 버튼으로 연주

**Part 4**: 소리로 표현하기 - 부저 활용

---

## 학습 목표

- [ ] 여러 버튼 입력을 동시에 처리할 수 있다
- [ ] 버튼별로 다른 음을 출력하는 프로그램을 작성할 수 있다
- [ ] 이전에 배운 내용들을 조합하여 복합 프로젝트를 완성할 수 있다

## 개요

버튼을 누르면 해당하는 음이 나오는 미니 피아노를 만들어봅시다! 4개의 버튼에 '도, 레, 미, 파'를 할당하면 손가락으로 직접 연주할 수 있습니다.

이 프로젝트는 지금까지 배운 버튼 입력, 조건문, 부저 제어를 모두 종합합니다. LED를 추가해서 누른 버튼에 해당하는 LED가 함께 켜지게 하면 더 화려한 악기가 완성됩니다. 친구와 함께 이중주에 도전해보세요!

## 예상 소요 시간

2시간
