# 삐- 소리의 원리 - 부저 연결과 PWM

**Part 4**: 소리로 표현하기 - 부저 활용

## 학습 목표

- [ ] 피에조 부저의 동작 원리를 설명할 수 있다
- [ ] PWM(펄스 폭 변조)의 개념을 이해할 수 있다
- [ ] 주파수를 변경하여 다른 높이의 소리를 낼 수 있다

## 개요

피에조 부저는 전기 신호를 소리로 바꿔주는 작은 스피커입니다. 빠르게 진동하면 높은 소리, 느리게 진동하면 낮은 소리가 납니다. 이 진동 속도를 '주파수'라고 부릅니다.

PWM은 디지털 신호로 아날로그 같은 효과를 내는 기술입니다. LED 밝기 조절이나 모터 속도 제어에도 사용되지만, 여기서는 부저로 특정 주파수의 소리를 만드는 데 활용합니다. '도레미파솔라시도'를 직접 연주해봅시다!
