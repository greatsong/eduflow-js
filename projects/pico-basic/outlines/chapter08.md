# 누르면 반응해! - 조건문 활용

**Part 3**: 입력을 받아라 - 버튼과 상호작용

---

## 학습 목표

- [ ] if-else 조건문의 구조와 동작을 이해할 수 있다
- [ ] 버튼을 누르면 LED가 켜지는 프로그램을 작성할 수 있다
- [ ] 토글 기능(누를 때마다 상태 변경)을 구현할 수 있다

## 개요

조건문은 '만약 ~라면, ~해라'라는 논리를 코드로 표현하는 방법입니다. 버튼이 눌렸는지 확인하고, 눌렸다면 LED를 켜는 프로그램을 만들어봅시다.

한 단계 더 나아가서, 버튼을 누를 때마다 LED가 켜졌다 꺼졌다 하는 토글 기능도 구현해봅니다. 형광등 스위치처럼 한 번 누르면 켜지고, 다시 누르면 꺼지는 거죠. 이 과정에서 변수와 상태 저장의 개념도 자연스럽게 배울 수 있습니다.
