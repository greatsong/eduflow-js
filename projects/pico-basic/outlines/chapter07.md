# 버튼 연결하기 - 디지털 입력의 시작

**Part 3**: 입력을 받아라 - 버튼과 상호작용

---

## 학습 목표

- [ ] 푸시 버튼의 동작 원리를 이해할 수 있다
- [ ] 풀업/풀다운 저항의 필요성을 설명할 수 있다
- [ ] 버튼 상태를 읽어 화면에 출력할 수 있다

## 개요

지금까지는 피코가 일방적으로 명령을 내렸다면, 이제는 우리의 입력을 받을 차례입니다. 버튼은 가장 간단한 입력 장치로, 누르면 회로가 연결되고 떼면 끊어집니다.

버튼을 연결할 때는 '풀업' 또는 '풀다운' 저항이 필요합니다. 왜 필요한지 직접 실험해보면서 이해해봅시다. 피코 내부에 이미 저항이 내장되어 있어서 코드 한 줄로 쉽게 사용할 수 있답니다!
