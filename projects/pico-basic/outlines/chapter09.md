# 반응속도 게임 만들기

**Part 3**: 입력을 받아라 - 버튼과 상호작용

---

## 학습 목표

- [ ] 시간 측정 함수를 사용해 경과 시간을 계산할 수 있다
- [ ] 랜덤 함수를 활용해 예측 불가능한 동작을 구현할 수 있다
- [ ] 버튼과 LED를 조합한 간단한 게임을 완성할 수 있다

## 개요

LED가 켜지는 순간 버튼을 최대한 빨리 누르는 반응속도 게임을 만들어봅시다! 랜덤한 시간 후에 LED가 켜지고, 버튼을 누르면 반응 시간이 측정됩니다.

이 프로젝트에서는 랜덤 함수와 시간 측정 함수를 새로 배웁니다. 완성된 게임으로 친구들과 대결해보세요. 누가 가장 빠른 반응속도를 가졌는지 겨루는 것도 재미있고, 기록을 갱신하는 재미도 쏠쏠합니다!
