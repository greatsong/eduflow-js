# Hello, Pico! - 첫 번째 프로그램 실행

**Part 1**: 피코와 첫 만남

---

## 학습 목표

- [ ] Thonny에서 간단한 파이썬 코드를 작성하고 실행할 수 있다
- [ ] print() 함수로 메시지를 출력할 수 있다
- [ ] 피코의 내장 LED를 켜고 끌 수 있다

## 개요

드디어 첫 번째 코드를 실행할 시간입니다! 먼저 화면에 'Hello, Pico!'를 출력하는 가장 간단한 프로그램으로 시작합니다. 코드 한 줄로 내가 원하는 메시지가 화면에 나타나는 것을 보면, 프로그래밍의 첫 번째 성취감을 맛볼 수 있습니다.

그 다음엔 피코 보드에 내장된 작은 LED를 켜봅니다. 코드를 실행하는 순간 실제로 빛이 나는 것을 보면 '내가 정말 하드웨어를 제어하고 있구나!'라는 신기한 경험을 하게 될 거예요.
