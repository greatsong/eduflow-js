# 온도계 만들기 - 온도 센서 활용

**Part 5**: 세상을 감지하라 - 센서 활용

---

## 학습 목표

- [ ] 온도 센서의 동작 원리를 이해할 수 있다
- [ ] 센서 값을 실제 온도(섭씨)로 변환할 수 있다
- [ ] 측정값을 화면에 보기 좋게 출력할 수 있다

## 개요

피코에 내장된 온도 센서나 외부 온도 센서를 사용해 현재 온도를 측정해봅시다. 센서가 보내는 전압 값을 수식으로 변환하면 우리가 아는 섭씨 온도가 됩니다.

측정한 온도를 'XX.X°C' 형태로 예쁘게 출력하는 방법도 배웁니다. 온도에 따라 LED 색상을 바꾸거나, 특정 온도 이상이면 부저로 경고음을 울리는 기능을 추가해서 나만의 온도 경보기를 만들어보세요!
