## 하드웨어 환경

### 기본 장비
- Raspberry Pi Pico 2W (WiFi/Bluetooth 내장)
- Grove Shield for Pi Pico
- USB 케이블 (Micro USB)
- PC + Thonny IDE

### Grove 센서 (입력)
| 센서 | 용도 | 연결 포트 |
|------|------|----------|
| DHT11/DHT20 | 온습도 | D16 |
| 조도 센서 | 밝기 | A0 |
| 초음파 센서 | 거리 | D18 |
| MPU6050 | 가속도/자이로 | I2C |
| 버튼 | 입력 | D20 |
| 로터리 앵글 | 회전 입력 | A1 |

### Grove 출력 장치
| 장치 | 용도 | 연결 포트 |
|------|------|----------|
| LED | 단색 출력 | D16/D18 |
| RGB LED | 색상 출력 | D18 |
| 부저 | 소리 출력 | D20 |
| 서보모터 | 각도 제어 | D16 |
| OLED 0.96" | 화면 표시 | I2C |

## 코드 작성 원칙

### 주석 3단계
```python
# === 1단계: 무엇을 하는 코드인지 (WHAT) ===
# LED를 깜빡이는 코드예요

# --- 왜 필요한지 (WHY) ---
# 피코가 제대로 연결됐는지 확인하려면 LED를 켜봐야 해요

# 어떻게 동작하는지 (HOW)
from machine import Pin  # 핀 제어 라이브러리
import time              # 시간 관련 기능

led = Pin("LED", Pin.OUT)  # 내장 LED를 출력 모드로 설정
led.on()                   # LED 켜기
time.sleep(1)              # 1초 대기
led.off()                  # LED 끄기
```

### 단계별 파일 분리
- `step1_기본.py` → `step2_확장.py` → `step3_완성.py`
- 각 단계는 독립 실행 가능

### 복사 가능 원칙
- 학생이 바로 복사해서 실행 가능해야 함
- 수정이 필요한 부분은 명확히 표시 (예: `"여기에_와이파이_이름"`)
