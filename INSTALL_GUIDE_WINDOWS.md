# 에듀플로(EduFlow) 윈도우 설치 가이드

> AI가 교육자료를 자동으로 만들어주는 시스템입니다.
> 주제만 입력하면 **교재 목차 → 챕터 본문 → 웹사이트/워드 문서**까지 자동 생성됩니다.

---

## 전체 과정 요약

| 순서 | 할 일 | 소요 시간 |
|------|--------|----------|
| 1 | Git 설치 | 3분 |
| 2 | Node.js 설치 | 3분 |
| 3 | 에듀플로 다운로드 | 2분 |
| 4 | 실행 + API 키 입력 | 2분 |

**총 약 10분**이면 시작할 수 있습니다.

> API 키는 온라인 세션에서 제공해드립니다. 별도로 발급받을 필요 없습니다.

---

## 시작하기 전에

- **깃허브(GitHub) 계정**이 필요합니다 (이미 가입되어 있다고 가정합니다).
- **API 키**는 세션 시간에 채팅으로 보내드리니, 메모장에 복사해두세요.
- 아래 과정은 **세션 전에 미리 해두시면** 시간이 절약됩니다 (1~3단계).

---

## 1단계: Git 설치하기

Git은 프로그램 코드를 다운로드하기 위해 필요합니다.

### 1-1. 다운로드

1. 웹 브라우저(Chrome, Edge 등)에서 아래 주소로 이동합니다:

   👉 https://git-scm.com/download/win

2. 페이지가 열리면 **자동으로 다운로드**가 시작됩니다.
   - 만약 시작되지 않으면 **"Click here to download manually"** 를 클릭하세요.
   - **64-bit Git for Windows Setup** 을 선택하세요.

### 1-2. 설치

1. 다운로드된 파일(예: `Git-2.47.1-64-bit.exe`)을 **더블클릭**합니다.
2. 설치 화면이 나타나면 **모든 화면에서 "Next"만 계속 누릅니다.**
   - 아무 설정도 바꿀 필요 없습니다. 그냥 Next, Next, Next...
   - 마지막에 **"Install"** → **"Finish"** 를 누릅니다.

### 1-3. 설치 확인

1. 키보드에서 `Windows 키`를 누릅니다 (화면 왼쪽 아래 ⊞ 버튼).
2. **cmd** 라고 입력합니다.
3. **"명령 프롬프트"** 가 나타나면 클릭하여 엽니다.

> 💡 **명령 프롬프트란?** 글자로 컴퓨터에 명령을 내리는 검은 화면입니다. 무서워 보이지만 아래 명령어만 복사-붙여넣기 하면 됩니다!

4. 검은 화면에 아래를 **그대로 입력**하고 Enter를 누릅니다:

```
git --version
```

5. 아래와 비슷한 결과가 나오면 성공입니다:

```
git version 2.47.1.windows.1
```

> ⚠️ **"git을 찾을 수 없습니다"** 라고 나오면?
> → 명령 프롬프트를 닫고 **새로 열어서** 다시 해보세요.
> → 그래도 안 되면 **컴퓨터를 재시작**한 뒤 다시 해보세요.

## 2단계: Node.js 설치하기

Node.js는 에듀플로를 실행하기 위해 필요한 프로그램입니다.

### 2-1. 다운로드

1. 웹 브라우저에서 아래 주소로 이동합니다:

   👉 https://nodejs.org

2. 큰 초록색 버튼이 보입니다. **"LTS"** 라고 적힌 버튼을 클릭합니다.
   - LTS = 안정적인 버전이라는 뜻입니다.
   - 버전 숫자는 달라도 괜찮습니다 (20, 22 등).

### 2-2. 설치

1. 다운로드된 파일(예: `node-v22.13.0-x64.msi`)을 **더블클릭**합니다.
2. **모든 화면에서 "Next"만 계속 누릅니다.**
   - "I accept the terms" 체크박스가 있으면 **체크** ☑ 하고 Next.
   - 마지막에 **"Install"** → **"Finish"** 를 누릅니다.

### 2-3. 설치 확인

1. **명령 프롬프트를 새로 엽니다** (이전에 열어둔 것은 닫고 새로 열어야 합니다!).
   - `Windows 키` → "cmd" 입력 → 명령 프롬프트 클릭
2. 아래 명령어를 입력하고 Enter:

```
node --version
```

아래처럼 나오면 됩니다 (숫자는 다를 수 있음):
```
v22.13.0
```

3. 하나 더 확인합니다:

```
npm --version
```

아래처럼 나오면 됩니다:
```
10.9.2
```

> 두 개 다 숫자가 나오면 성공입니다! 숫자가 조금 달라도 괜찮습니다.

## 3단계: 에듀플로 다운로드하기

### 3-1. 명령 프롬프트에서 바탕화면으로 이동

명령 프롬프트가 열려 있는 상태에서, 아래를 입력하고 Enter:

```
cd %USERPROFILE%\Desktop
```

> 이렇게 하면 바탕화면 폴더로 이동한 것입니다.

### 3-2. 에듀플로 다운로드

아래 명령어를 **그대로** 입력하고 Enter:

```
git clone https://github.com/greatsong/eduflow-js.git
```

아래와 같이 진행 상황이 표시됩니다:
```
Cloning into 'eduflow-js'...
remote: Enumerating objects: ...
Receiving objects: 100% ...
done.
```

완료되면 바탕화면에 **`eduflow-js`** 라는 폴더가 생깁니다!

### 3-3. 에듀플로 폴더로 이동

```
cd eduflow-js
```

### 3-4. 패키지 설치

```
npm install
```

처음 실행하면 **1~3분** 정도 걸립니다. 많은 텍스트가 주르르 흘러갑니다. **정상입니다!**

아래와 비슷한 메시지가 나오면 완료:
```
added 300 packages in 45s
```

> 💡 **"warn"** 이라는 노란색 경고가 나와도 괜찮습니다.
> **"error"** 라는 빨간색 오류만 아니면 됩니다.

## 4단계: 실행 + API 키 입력

### 4-1. 에듀플로 시작

명령 프롬프트에서 아래를 입력합니다:

```
npm run dev
```

아래와 같은 메시지가 나오면 성공입니다:

```
[0] [EduFlow] 서버 실행 중: http://localhost:7829
[1]   VITE v6.x.x ready in xxx ms
[1]   ➜  Local:   http://localhost:7830/
```

### 4-2. 브라우저에서 열기

1. 웹 브라우저(Chrome, Edge 등)를 엽니다.
2. **주소창**에 아래를 입력하고 Enter:

```
http://localhost:7830
```

> ⚠️ **주소창**에 입력하세요! 구글 검색창이 아닙니다!
> 주소창은 브라우저 맨 위에 있는 긴 입력 칸입니다.

### 4-3. API 키 입력하기

> 🔑 세션에서 공유드리는 API 키를 준비해주세요.

1. 에듀플로 화면 왼쪽 아래에 **🔑 API 키 필요** 버튼이 보입니다. 클릭합니다.
2. 팝업창에 세션에서 받은 **API 키를 붙여넣기** (`Ctrl + V`)
3. **"저장"** 버튼을 클릭합니다.
4. "검증 중..." 이후 팝업이 닫히면 완료! 버튼이 **✅ API 키 설정됨**으로 바뀝니다.

> 💡 API 키는 **브라우저에 자동 저장**됩니다. 다음에 다시 입력할 필요 없습니다.
> 파일을 편집할 필요 없이 웹 화면에서 바로 설정할 수 있습니다!

**에듀플로가 정상 작동하면 설치 완료입니다!**

## 에듀플로 사용법

### 교재 만들기 (6단계 워크플로)

에듀플로는 **6단계**를 따라가며 교재를 만듭니다:

```
📁 프로젝트 관리  →  💬 방향성 논의  →  📋 목차 작성
                                            ↓
🚀 배포 관리    ←   ✍️ 챕터 제작   ←   ✅ 피드백 컨펌
```

#### 1) 프로젝트 관리 (시작점)

**"프로젝트 관리"** 를 클릭하고, 프로젝트를 만듭니다:

| 입력 항목 | 예시 |
|----------|------|
| 프로젝트 이름 | `파이썬-입문` (공백 대신 `-` 사용) |
| 저자 | `홍길동` |
| 설명 | `초등학생을 위한 파이썬 기초 교재` |

**"프로젝트 생성"** 버튼을 클릭합니다.

> 💡 **참고자료**가 있으면 "참고자료" 탭에서 PDF나 텍스트 파일을 업로드할 수 있습니다.
> AI가 이 자료를 참고해서 교재를 만듭니다.

#### 2) 방향성 논의

왼쪽 메뉴에서 **"방향성 논의"** 를 클릭합니다.

AI와 채팅으로 교재의 방향을 정합니다:
- "초등학교 5~6학년을 대상으로 파이썬 기초를 재미있게 가르치는 교재를 만들고 싶어요"
- "게임 만들기를 예제로 활용하고 싶어요"
- "각 챕터마다 실습 문제가 있었으면 좋겠어요"

AI가 질문하면 답변하고, 충분히 이야기했으면 다음 단계로 넘어갑니다.

#### 3) 목차 작성

**"목차 작성"** 을 클릭하고 **"생성"** 버튼을 누릅니다.

AI가 앞서 나눈 대화를 바탕으로 **Part > Chapter 구조의 목차를 자동 생성**합니다.
각 챕터에는 제목, 학습 목표, 개요가 포함됩니다.

마음에 안 드는 부분이 있으면 "편집" 탭에서 수정할 수 있습니다.

#### 4) 피드백 컨펌

**"피드백 컨펌"** 에서 목차를 최종 검토합니다.

AI와 대화하며 수정할 부분을 고치고, 만족스러우면 **"목차 확정"** 을 누릅니다.

> 확정 후에는 목차를 수정하기 어려우니, 꼼꼼히 확인하세요!

#### 5) 챕터 제작

**"챕터 제작"** 이 핵심 단계입니다.

**"배치 자동화"** 탭을 선택하면 모든 챕터를 **한 번에 자동 생성**할 수 있습니다:

1. **모델 선택** - 아래 가이드 참고
2. **"생성 시작"** 클릭
3. AI가 챕터를 하나씩 만듭니다 (진행률이 실시간 표시됨)
4. 완료되면 비용 리포트가 나옵니다

> 💡 10챕터 기준으로 약 5~30분 정도 걸립니다 (모델에 따라 다름).

**"챕터 편집"** 탭에서 완성된 챕터 내용을 확인하고 수정할 수 있습니다.

#### 6) 배포 관리

완성된 교재를 내보내는 단계입니다:

| 배포 방식 | 설명 | 추가 설치 |
|----------|------|----------|
| **미리보기** | 브라우저에서 바로 확인 | 없음 |
| **DOCX** | 워드 파일로 다운로드 | Pandoc 필요 |
| **MkDocs** | 웹사이트로 만들기 | MkDocs 필요 |

> 처음에는 **미리보기**만으로도 충분합니다!

### AI 모델 선택 가이드

| 모델 | 속도 | 품질 | 10챕터 비용 | 추천 상황 |
|------|------|------|-----------|----------|
| **Haiku 4.5** | ⚡⚡⚡ 매우 빠름 | ★★★☆ 괜찮음 | ~300원 | 빠르게 테스트할 때 |
| **Sonnet 4.6** | ⚡⚡ 빠름 | ★★★★ 좋음 | ~2,000원 | 일반적으로 추천 |
| **Opus 4.6** | ⚡ 느림 | ★★★★★ 최고 | ~7,000원 | 중요한 교재 |

> 💡 **팁:** 처음에는 **Haiku**로 빠르게 테스트해보고, 결과가 만족스러우면 **Sonnet**이나 **Opus**로 다시 만드는 것을 추천합니다.

## 다음에 다시 시작하는 방법

컴퓨터를 껐다가 다시 켰을 때:

### 방법 1: 명령 프롬프트에서 (기본)

1. `Windows 키` → "cmd" → 명령 프롬프트 열기
2. 아래 명령어 입력:

```
cd %USERPROFILE%\Desktop\eduflow-js
npm run dev
```

3. 브라우저에서 `http://localhost:7830` 접속

### 방법 2: 바로가기 만들기 (더 편리!)

한 번만 만들어두면 다음부터는 **더블클릭 한 번**으로 시작할 수 있습니다.

1. 바탕화면에서 **마우스 우클릭** → **새로 만들기** → **바로 가기**
2. "항목 위치 입력" 란에 아래를 **그대로** 붙여넣습니다:

```
cmd /k "cd %USERPROFILE%\Desktop\eduflow-js && npm run dev"
```

3. **"다음"** 클릭
4. 이름을 **에듀플로 실행** 으로 입력
5. **"마침"** 클릭

이제 바탕화면의 **"에듀플로 실행"** 아이콘을 더블클릭하면 자동 시작됩니다!
(시작되면 브라우저에서 `http://localhost:7830` 을 열면 됩니다)

### 종료 방법

명령 프롬프트(검은 화면)에서 `Ctrl + C`를 누릅니다.
"일괄 작업을 끝내시겠습니까?" 라고 나오면 `Y`를 누르고 Enter.

## 자주 묻는 질문 (FAQ)

### Q1. "API 키가 필요합니다" 또는 AI 기능이 안 돼요

API 키가 설정되지 않았거나 만료된 것입니다.

1. 에듀플로 화면 왼쪽 아래 **🔑 API 키 필요** 버튼 클릭
2. 세션에서 받은 API 키를 붙여넣고 **"저장"**
3. 버튼이 **✅ API 키 설정됨**으로 바뀌면 정상

"유효하지 않은 API 키" 메시지가 나오면 키를 다시 확인해주세요.

### Q2. "port already in use" 오류가 나요

이전에 실행한 에듀플로가 아직 실행 중인 것입니다.

1. 명령 프롬프트를 **새로** 엽니다.
2. 아래를 입력합니다:
```
taskkill /F /IM node.exe
```
3. 원래 명령 프롬프트로 돌아가서 `npm run dev`로 다시 실행합니다.

### Q3. npm install에서 오류가 나요

1. Node.js가 설치되어 있는지 확인합니다:
```
node --version
```
   숫자가 안 나오면 **2단계**로 돌아가서 Node.js를 설치하세요.

2. 숫자가 나오는데 install이 안 되면, 캐시를 지우고 다시 시도합니다:
```
npm cache clean --force
npm install
```

### Q4. 브라우저에서 화면이 안 나와요

1. 명령 프롬프트에 `http://localhost:7830` 이 표시되어 있는지 확인합니다.
   - 표시가 안 되어 있으면 아직 로딩 중이니 **30초** 정도 기다려보세요.
2. 주소를 브라우저의 **주소창**에 직접 입력합니다.
   - ⚠️ **구글 검색창**에 입력하면 안 됩니다! 브라우저 **맨 위** 주소 입력 칸에 넣으세요.
3. "Windows 방화벽에서 차단했습니다" 팝업이 나오면 **"허용"** 을 클릭하세요.

### Q5. 만든 교재는 어디에 저장되나요?

바탕화면의 `eduflow-js` 폴더 안 `projects` 폴더에 자동 저장됩니다:

```
바탕화면/
  └── eduflow-js/
      └── projects/
          └── 파이썬-입문/        ← 내 프로젝트
              ├── config.json     (설정)
              ├── toc.json        (목차)
              ├── docs/           (챕터 내용들)
              └── discussions/    (AI 대화 기록)
```

이 `projects` 폴더를 USB에 복사하면 **백업**할 수 있습니다.

### Q6. 에듀플로를 업데이트하고 싶어요

새 기능이 추가되면 아래 명령어로 업데이트합니다:

```
cd %USERPROFILE%\Desktop\eduflow-js
git pull
npm install
npm run dev
```

### Q7. 워드(DOCX) 파일로 내보내고 싶어요

워드 내보내기를 쓰려면 **Pandoc**이라는 프로그램이 추가로 필요합니다:

1. 아래 주소에서 Windows 버전을 다운로드합니다:
   👉 https://pandoc.org/installing.html
2. 설치 후 명령 프롬프트를 **새로** 열고, 에듀플로를 재시작합니다.
3. "배포 관리" → "DOCX" 탭에서 사용할 수 있습니다.

> 워드 내보내기가 필요 없으면 Pandoc은 설치하지 않아도 됩니다.

처음 보는 단어가 있을 수 있어서 정리했습니다.

| 용어 | 뜻 |
|------|-----|
| **Git** | 코드를 다운로드하는 도구 (1단계에서 설치) |
| **Node.js** | 에듀플로를 돌리는 엔진 (2단계에서 설치) |
